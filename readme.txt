Hola lector!
Este proyecto se trata de una agencia de viajes en latam, llamada CAPRI.
El objetivo de este sito web es crear una interfaz simple y practica, para que el usuario no tenga problemas al encontrar la información que quiere.
Nuestro sitio contiene 5 paginas html.
Todas las imagenes tienen sus ALT completos con información.
Todas las paginas html tienen un title unico, meta desciption y keywords.

Keywords:
"viajes, destinos, promociones, turismo, viajar, seguro"


Meta description utilizadas: 

index.html:"Agencia de viajes, destinos, promociones, viajes nacionales, internacionales"

 destinos.html:"Destinos para viajar, Argentina, Brasil y Uruguay."

nosotros.html:"Capri agencia de viajes, viajes seguros y en promocion, información sobre la empresa y su historia"

promociones.html:"Viajes en promociones para Argentina"

agencias.html:"Agencias de viajes, en Brasil, Argentina y Uruguay"
