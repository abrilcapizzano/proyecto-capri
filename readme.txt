Hola lector!
Este projecto se trata de una agencia de viajes en latam, llamada CAPRI.
El objetivo de este sito web es crear una interfaz simple y practica, para que el usuario no tenga problemas al encontrar la información que quiere.
Nuestro sitio contiene 5 paginas html.
Todas las imagenes tienen sus ALT completos con información.
Todas las paginas html tienen un title unico, meta desciption y keywords.
